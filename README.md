# Vibecoding i praksis – Webinar Landing Page

Offisiell landingside for webinaret **Vibecoding i praksis** – arrangert av **EMP Secure** og **Arrow** den **26. mars 2026**.

## Om webinaret

Dette webinaret viser hvordan du får flyt og effektivitet i kodeutviklingen ved hjelp av **GitHub Copilot**, **VS Code** og moderne AI-assistert programmeringspraksis.

Vi demonstrerer vibecoding live – fra idé til produksjonsklar kode – og dykker ned i:
- **Hva vibecoding faktisk betyr** i praksis
- **GitHub Copilot som AI-partner** i daglig utvikling
- **GitHub som platform** for skalert samarbeid
- **Enterprise-løsninger** med GitHub Enterprise og Foundry
